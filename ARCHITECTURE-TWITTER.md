## Key Components

### 1. Client (`client.ts`)
- Wraps `twitter-api-v2` library
- Provides clean API for all operations
- Handles authentication
- Error handling and retries

### 2. Relay (`relay.ts`)
- Polling loop (configurable interval)
- Monitors DMs and mentions
- User filtering
- State tracking
- Graceful error recovery

### 3. Auto-Reply (`auto-reply.ts`)
- Two modes: static / command
- Session management
- Command execution with timeout
- Reset trigger handling
- Response formatting

### 4. Commands (`commands/twitter-*.ts`)
- CLI interface
- Option parsing
- Client coordination
- Output formatting

### 5. Main CLI (`twitter-agent.ts`)
- Command registration
- Environment loading
- Client initialization
- Help documentation
